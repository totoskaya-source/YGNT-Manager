# YGNT Manager - Idées post-V1

Ce document recense uniquement les idées d'amélioration prévues après la
sortie de la V1 afin de préserver la stabilité et la finalisation de la
version 1.0.

Ce document est volontairement séparé de la ROADMAP. Il ne contient aucune
fonctionnalité obligatoire pour la V1 : il sert uniquement de backlog pour
les versions futures.

---

# Workflow

- Après la création d'un Devis, rester dans le dialogue afin de générer
  immédiatement le DOCX/PDF.
- Même comportement pour les Contrats.
- Même comportement pour les Factures.
- Bouton "Ouvrir le dossier des documents".
- Bouton "Dupliquer" (Prestations, Devis, Contrats, Factures).
- Raccourcis clavier.
- Amélioration des messages de confirmation.
- Ouverture plus fluide des documents.

# Outils

Créer un menu "Outils".

Prévoir notamment :

- Simulateur IntermiPaie (ouverture directe du simulateur officiel).
- Préparation des cachets avant ouverture d'IntermiPaie.
- Répartition des cachets.
- Calcul HT ↔ TTC.
- Calcul des frais kilométriques.

Ne jamais reproduire le moteur de calcul d'IntermiPaie sans source
officielle.

Si une API officielle existe un jour, prévoir son intégration.
